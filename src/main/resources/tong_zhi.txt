● 根据代码，通知的 type 和 link 含义如下：
  type — 通知类型
  用于区分通知的类别，决定显示的图标和颜色。共有 5 种取值：                                                                                                                                                                                                                                                     ┌──────────────────┬────────────────┬───────────────┬──────┐
  │        值        │      含义      │     图标      │ 颜色 │
  ├──────────────────┼────────────────┼───────────────┼──────┤
  │ disease_alert    │ 病害预警       │ WarningFilled │ 红色 │
  ├──────────────────┼────────────────┼───────────────┼──────┤
  │ treatment_remind │ 施药提醒       │ Clock         │ 橙色 │
  ├──────────────────┼────────────────┼───────────────┼──────┤
  │ safety_interval  │ 安全间隔期提醒 │ CircleCheck   │ 绿色 │
  ├──────────────────┼────────────────┼───────────────┼──────┤
  │ weather_alert    │ 天气预警       │ Cloudy        │ 蓝色 │
  ├──────────────────┼────────────────┼───────────────┼──────┤
  │ system           │ 系统通知       │ Bell          │ 灰色 │
  └──────────────────┴────────────────┴───────────────┴──────┘

  type 同时也用于筛选栏的过滤功能（NotificationIndex.vue:17-24）。

  link — 跳转路由

  用户点击通知时，如果 link 非空，会调用 router.push(item.link) 跳转到对应页面（NotificationIndex.vue:98-100）。

  从 mock 数据来看，link 的值是前端路由路径，例如：

  - /knowledge/disease/wheat_rust — 跳转到知识库中具体病害页
  - /diagnosis — 跳转到诊断页
  - /farm — 跳转到农场管理页
  - /knowledge — 跳转到知识库首页
  - ''（空字符串）— 不跳转，仅查看通知内容
